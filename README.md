# Giskard Core

Core shared utilities and foundational components for the Giskard library ecosystem.

## Installation

```bash
uv add giskard-core
# or
pip install giskard-core
```

## Requirements

- Python >= 3.12
- pydantic >= 2.11.0, < 3

## Development

```bash
make setup    # Install dependencies
make test     # Run tests
make lint     # Run linting
make format   # Format code
```
